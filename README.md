# Off-Grid Inverter System – MATLAB/Simulink

## Overview
Design and simulation of an off-grid inverter system using MATLAB/Simulink
and Simscape Electrical.

The model includes a DC–DC step-up stage, a PWM-controlled H-bridge inverter,
LC output filtering, and performance monitoring.

## Simulink Model
![Simulink Model](media/inverter_simulink_model.png)

## Features
- DC–DC boost converter
- PWM-controlled MOSFET H-bridge inverter
- LC output filter
- Voltage and current monitoring
- Efficiency calculation
- THD analysis

## Tools
- MATLAB
- Simulink
- Simscape Electrical
